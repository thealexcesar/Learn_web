Tutorial básico de Linux

    Apt: o gerenciador de pacotes do Ubuntu
  1. Verifique se o programa já está instalado:
    $ (nome-programa) --version
  2. Atualizar:
    $ sudo apt update
  3. Instalar:
    $ sudo apt install
  4. Deletar:
    $ sudo apt purge

  5. Quando o programa não está incluso no apt, tem que ser baixado manualmente.

    Exemplo, baixando google-chrome: Faça o download no site depois vá até a pasta do arquivo, renomeie-o como chrome.deb clique com o botão direito na pasta do areuivo e tecle em abrir com terminal, e digite o comando:
    $ sudo dpkg -i chrome.deb
